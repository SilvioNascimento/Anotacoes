- SMS -
	* Quanto custa, para uma operadora de telefonia móvel, enviar um SMS?
		- 0. Nada. Vazio. Null
	* O celular troca, naturalmente, alguns bits com as torres de comunicação.
		- Ex: "Você está aí?" "Estou" "Você está aí?" "Estou" "Você está aí?" "Estou"

- MMS -
	* Transmissão de mensagens multimídia (áudio, vídeo) por meio de uma espécie de conexão de dados primitiva

- Conexões móveis -
	* 1G (analógico): 10Kbps (10 kilobits por segundo)
	* 2G (digital) (GSM (Global System for Mobile Communication)): 97 Kbps
	* GPRS (General Packet Radio Service) / "2,5G": 32-80Kbps - dados + voz
	* EDGE (Enhanced Data Rates for GSM Evolution) / "2,75G": 128-236 Kbps
	* 3G (7 Mbps), 4G (22,1Mbps), 5G (10Gbps)
