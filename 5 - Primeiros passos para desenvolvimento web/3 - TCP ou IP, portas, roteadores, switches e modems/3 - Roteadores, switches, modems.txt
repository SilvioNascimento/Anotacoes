- Modem -
	* MOdulator - DEModulator
	* Hardware que converte dados em um formato que possa ser transmitido como sinal de um computador para outro e lido por outro (Hardware que modula e demodula sinais; Dados que os humanos compreendem enquanto os computadores não)

- Roteador -
	* Distribui internet para um ou mais dispositivos de uma rede
	* Pode fazer a comunicação entre redes
	* Pode ser "burro" (Não sabe se todo mundo está recebendo e não sabe quem está recebendo)

- Switch -
	* Distribui internet para um ou mais dispositivos de uma rede
	* Criado para ser "inteligente" (sabe se todo mundo está recebendo e sabe quem está recebendo)
